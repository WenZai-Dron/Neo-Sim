# 游戏根目录文件结构
```
{游戏根目录}/
└── NeoSim/
    ├── Skins/                                  # 皮肤目录
    ├── Buildings/                              # 建筑目录
    └── data/                                   # 数据目录
        │
        ├── {saveName}/                         # [非服务器]按世界存档名隔离
        │   ├── data.json                       # 世界级存档数据
        │   │
        │   └── {cityName}/                     # 城市目录
        │       ├── data.json                   # 城市数据
        │       │                               #   { mode, population, dayOfWeek, day, credit,
        │       │                               #     runGuiSent, joinedPlayers }
        │       ├── player.json                 # 城市玩家列表
        │       │                               #   { "players": ["player1", "player2", ...] }
        │       └── npc/                        # NPC目录
        │           └── {npcName}.json          # 单个NPC存档文件
        │                                       #   { name, surname, givenName, sex, skin,
        │                                       #     uuid, position: {x,y,z}, yaw, pitch,
        │                                       #     health, maxHealth }
        │
        ├── data.json                           # [服务器]全局数据
        │
        └── {cityName}/                         # [服务器]
            ├── data.json
            ├── player.json
            └── npc/
                └── {npcName}.json
```
## 各文件说明
| 文件/目录 | 创建时机 | 用途 |
|---|---|---|
| `NeoSim/` | 服务器启动时 (`ServerStartingEvent`) | 根目录 |
| `NeoSim/Skins/` | 服务器启动时 | 自定义NPC皮肤存放处 |
| `NeoSim/Buildings/` | 服务器启动时 | 建筑数据（预留） |
| `NeoSim/data/` | 服务器启动时 | 数据根目录 |
| `data.json` | 服务器首次加载 `ModSavedData` 时 | [服务器] 全局存档数据 |
| `{cityName}/` | 玩家在City界面创建/选择城市时 | 城市数据隔离 |
| `{cityName}/data.json` | 城市创建时 | 城市存档（mode/population/day/credit） |
| `{cityName}/player.json` | 玩家加入城市时 | 该城市下的玩家白名单 |
| `{cityName}/npc/` | 城市创建时 | NPC数据目录 |
| `{npcName}.json` | NPC生成/保存时 | NPC持久化数据 |
## 相关代码文件
| 文件 | 职责 |
|---|---|
| `storage/FileCreater.java` | 目录创建、player.json 读写、城市查找 |
| `storage/ModSavedData.java` | data.json 读写、网络同步 |
| `storage/SimData.java` | 数据模型 record (mode, population, dayOfWeek, day, credit) |
| `npc/npcData.java` | NPC JSON 文件的保存/加载/删除 |
